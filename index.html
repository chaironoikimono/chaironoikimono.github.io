sync & backup

stores your log as a secret gist in your own GitHub account, so every device shows the same data and you've got an off-device copy. paste a classic token with only the gist scope checked (it can't touch anything else in your account). the token is saved on this device only — never in the app's code.

heads-up: a secret gist isn't listed or searchable, but it's reachable by anyone who has the exact gist URL — it's obscure, not encrypted. fine for a weight log; if you want it fully private later, I can switch the backend to a private repo instead.

regimes

a regime counts days from its start, like cycle day — e.g. reta d12. it shows as a badge on every log on/after the start, and a dashed line on the chart. leave the end date blank while it's ongoing.
